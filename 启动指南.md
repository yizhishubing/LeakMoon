# 启动指南

> 电脑重启后，只需一步即可启动整个项目，无需手动配置端口。

---

## 一键启动（推荐）

### Windows（双击即可）

```
直接双击项目根目录的 start.bat
```

会自动：
1. 检查 MySQL 和 Redis 是否运行，未运行则自动启动
2. 后端自动寻找空闲端口启动（默认 8000，被占用则顺延至 8001、8002...）
3. 前端在 5173 端口启动
4. 弹出两个命令行窗口，分别显示前后端日志

### Git Bash / WSL

```bash
cd d:/intern/LeakMoon
bash start.sh
```

## 单独启动

### 后端

```bash
cd d:\intern\LeakMoon\backend
venv\Scripts\activate
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

> 如果 8000 被占用，改为 `--port 8001` 或直接 `python -m app.main`（会自动找空闲端口）。

### 前端

```bash
cd d:\intern\LeakMoon\frontend
npm run dev
```

## 端口自动切换原理

后端 `app/main.py` 内置了 `find_free_port()` 函数：

```python
def find_free_port(start_port: int = 8000, end_port: int = 9000) -> int:
    """从 start_port 开始试探，找到第一个空闲端口"""
    for port in range(start_port, end_port + 1):
        with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
            try:
                s.bind(("0.0.0.0", port))
                return port
            except OSError:
                continue
```

**工作流程：**

```
启动后端
  │
  ▼
试探 8000 端口
  │
  ├─ 空闲 → 使用 8000 ✅
  │
  └─ 被占用 → 试探 8001
                │
                ├─ 空闲 → 使用 8001 ✅
                │
                └─ 被占用 → 试探 8002 → ... → 8010
```

**重启后不需要做任何配置**，直接 `start.bat` 即可。

## 常见问题

### Q: 启动后提示"后端未连接"？
A: 检查后端窗口输出的端口号，如果显示 `Starting on port 8001`，说明 8000 被占用了，前端需要对应修改。

### Q: MySQL/Redis 启动失败？
A: 确保已安装并在服务中启用：
```bash
# 启动 MySQL
net start MySQL84
# 启动 Redis（Windows 服务）
"C:\Program Files\Redis\redis-server.exe" --service-start
```

### Q: 如何彻底关闭所有服务？
A: 关闭所有弹出的命令行窗口即可，或双击 `start.bat` 后按 Ctrl+C。
